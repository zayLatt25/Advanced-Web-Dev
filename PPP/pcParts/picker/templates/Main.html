API Endpoints

Below are the available API endpoints and functionalities:

  • GET /api/pc_assembly/
    Access the PC Assembly API to retrieve data.
  • POST /api/save-customizations/
    Save customizations for a PC build.
  • DELETE /api/delete-customization/<saved_id>
    Delete a saved customization by ID. (Example ID: 1)

Unit Test and Load Data

To run unit tests or load data, use the following commands:

  • Run Unit Tests:

    • python manage.py test

  • Load Data:

    • python manage.py load_data

    The load_data command loads the JSON files under the /management/commands directory into the database.
